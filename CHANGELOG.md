# Changelog

All notable changes to the Ecclesia Dev website will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/).

## [0.1.0] - 2026-03-03

### Added
- Dark theme, minimal text-first homepage design
- Liturgical color theme synced to 1962 Roman calendar
- Two-column desktop layout for homepage
- Rome and Holy Land pilgrimage field guides
- Sacramentals articles
- Deploy script and Makefile target
- mtree file integrity verification article
- GitHub links on project names

### Fixed
- Advent November days and Paschaltide Sundays calendar logic
- color-mix fallback and removed dead --gold CSS vars
- Hero size reduced, body max-width widened
- Deploy chown updated for non-root deploy user
- Bernadette congregation, First Saturday, and Crawley-Boevey theology corrections
- Confession window corrected to ~20 days per Enchiridion
- Agent team credit removed from footer
